Date and Time : Wednesday, June 27, 2018 • 11:30 a.m. IST • 4 p.m. AEST • 3 p.m. JST • 2 p.m. CST • 2 p.m. SGT • 3 p.m. KST • 6 p.m. NZST

Duration : 60 Minutes

With rhel7.5, we introduced the virtual data optimizer (VDO). This software can help to save disk space. VDO sits on top of any block storage devices and provides zero-block elimination, deduplication of redundant blocks, and data compression. These pieces allow VDO to reduce data footprint on storage.

Join this webinar to learn about:
- The basics of VDO
- Use cases: where can VDO help?
- VDO setup and configuration
- How is VDO influencing read/write performance?
- How much storage space can I save for my use cases?

Speaker: Christian Horn, Technical Account Manager, Red Hat Tokyo.
