APAC TAM Webinar Title: : Limited Guide to HotSpot Garbage Collection
Date and Time: 6/13 오후 3시 (한국시간 ) / Tuesday June 13, 2017 • 4 p.m. AEST

Garbage Collection (GC) is an indispensable part of HotSpot Java Virtual Machine (JVM) including Oracle JDK and OpenJDK. It has been battle-tested for many years and evolved to having several GC algorithms. In this guide, we will learn teh basic idea about HotSpot GC and go through each of important GC algorithms. Then notice some important bug fixes and learn how to explore the source code.

Join this webinar to learn about:

Characteristics of HotSpot GC
Useful JVM options
Generations
Parallel, CMS, and G1 collectors
Recent bug fixes
Source code exploration using GNU Global

Speaker

Osamu Nagano (Middleware Software Maintenance Engineer)
