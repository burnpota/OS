systemctl daemon-reload

systemctl enable lvm2-activation-early.service
systemctl enable lvm2-activation.serviceApply #the modification to the 10-dm.rules file

안녕하세요,
Red Hat 한진구 입니다.

우선 현재 적용된 lvm filter 부분의 regular expression이 잘못 설정된것으로 인하여 발생했던 것으로 보입니다.
아래와 같이 수정한 후, dracut을 다시 만들고 시스템 리부팅 후에 다시 모니터링 바랍니다.

# vi /etc/lvm/lvm.conf

[...]

filter = [ "a|/dev/mapper/erp_ap_.*|", "a|/dev/sda[1-9]|", "a|/dev/sdb[1-9]|", "r|.*|" ]


dracut은 수행전에 이전 initramfs를 백업해두시고 작업하시기 바랍니다.

관련문서 첨부드립니다.
- https://access.redhat.com/node/1958
