안녕하세요,

Red Hat Global Support Service 를 이용해주셔서 감사합니다.

아래의 참고문서[1]과 아래의 예제에서 보는것과 같이 acronis 디바이스를 /etc/multipath.conf 의 blacklist 섹션에 추가하고,

blacklist {
devnode "^snumbd1d" # ---> 디바이스명은 반드시 재확인 필요.
}

$ service multipathd reload 만 수행을 해주시면 됩니다. 이 명령은 실제 온라인상으로도 실행이 가능하지만, multipathd 가
완전히 reload 되는데 매우 일시적인 지연이 있을수 있어서 만약 I/O 에 민감한 애플리케이션이 실행되는 환경이라면 가능한 부
하가 가장적은 시간대에 실행하시고, 정기유지보수 시점에 실행하는것도 시스템의 안정적인 운영이 좋은 방법일것입니다.


[ 참고문서 ]

[1] How to blacklist external hard drive taken by multipath
- https://access.redhat.com/solutions/68777

감사합니다.
