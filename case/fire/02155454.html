multipathd 의 경우 user space 의 어플리케이션이며, 예를 들어 말씀하신 /proc 와 /sys 같은 가상파일시스템의 경우

Show the currently used configuration, derived from default values and values specified within the configura‐tion file /etc/multipath.conf.
