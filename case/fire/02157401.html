안녕하세요,

Red Hat Global Support Service 를 이용해주셔서 감사합니다.

mod_proxy_hcheck 모듈은 RHSCL repo에 있는 httpd 2.4.25에서 제공되고 있습니다.

https://access.redhat.com/documentation/en-us/red_hat_software_collections/2/html/2.4_release_notes/chap-rhscl#sect-RHSCL-Changes-httpd

시스템이 레스햇 네트워크에 register되어 있을 경우 온라인으로 RHSCL repo를 활성화 하실수 있으십니다.
구체적인 방법:
1. 모든 repo 확인
# subscription-manager repos --list
rhel-x86_64-variant-7-rhscl-2.4 # repo 명칭은 이와 같은 포멧입니다.

2. 상기 확인된 repo를 활성화
# subscription-manager repos --enable rhel-x86_64-variant-7-rhscl-2.4

3. httpd 패키지 확인 및 설치
# yum search httpd24-httpd
# yum install httpd24-httpd

참고문서:
https://access.redhat.com/documentation/en-us/red_hat_software_collections/3/html/3.0_release_notes/chap-installation#sect-Installation-Subscribe

만일 오프라인으로 사용중인 시스템이시다면 아래의 링크에서 iso 파일을 다운로드 받으셔서 시스템에 마운트 하신후
관련 패키지를 설치하실수 있으십니다.
https://access.redhat.com/downloads/content/201/ver=1/rhel---7/2.4/x86_64/product-software

감사합니다.
