안녕하세요,

Red Hat Global Technical Support 서비스를 이용해주셔서 감사합니다.

2번 문의 사항에 대해서 업데이트 드립니다.

확인 결과 httpd worker MPM 사용을 위한 설정 및 모듈 정보에 관해서 자세한 내용을 Red Hat Knowledgebase article [1]에서 자세히 참조 하실 수 있습니다.

RHEL 7.x 의 경우 /etc/httpd/conf.modules.d/00-mpm.conf 설정 파일에서
LoadModule mpm_worker_module modules/mod_mpm_worker.so
을 enable 해야 합니다. modules/mod_mpm_worker.so 모듈 파일은 이미 설치되어 있으며 따로 추가 설치는 필요없습니다.

전체 상세한 내용을 article [1]에서 참고하시기 바랍니다.

만약 본 케이스와 관련하여 추가 질문이 있으시다면 알려주십시오. 또는 더 이상 지원이 필요하지 않다면 케이스를 close 할 수 있도록 알려주시기 바랍니다.

감사합니다.

[1] How to switch the MPM used in Apache httpd 2.4 https://access.redhat.com/solutions/2063063
How do I run Apache httpd with the worker mpm? https://access.redhat.com/solutions/25010

답변이 늦어져서 죄송합니다. 먼저 1번 문의에 대해서 업데이트 드립니다.

>> 1. 현재 httpd 버전이 2.4.6 에서 마이너 패치만 되는데 2.4.6 이상 버전이 제공되는지요?(예를 들어 2.4.31 이상)

Red Hat Knowledgebase article [1]의 내용과 같이 RHEL 7에서는 2.4.6 버전까지만 제공됩니다. 가장 최신 Apache 버전은 Red Hat JBoss Core Services (JBCS)5 Apache HTTP Server 2.4.29 입니다. 참고로 JBCS 기술 지원을 받기 위해서는 JBoss Enterprise App Platform (EAP) subscription 이 필요합니다.

* Red Hat Enterprise Linux (RHEL)1 - httpd rpms
RHEL 7: based on upstream v2.4.6

* Red Hat JBoss Core Services (JBCS)5 Apache HTTP Server
Latest is based on upstream v2.4.29

2번 문의는 확인 중이며 확인이 완료되는데로 바로 업데이트하겠습니다.

감사합니다.

[1] Apache httpd versions supported in Red Hat Enterprise Linux https://access.redhat.com/solutions/445713
