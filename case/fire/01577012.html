Thank you for contacting Red Hat Technical Support.

My name is Jesus Serrano and I will be one of the engineers assisting with this case. I see you have problems with the following command:

pcs cluster setup --name dssa_cluster dssalp01-HB,dssalp01 dssalp02-HB,dssalp02
Warning: Unable to resolve hostname: dssalp01-HB,dssalp01
Warning: Unable to resolve hostname: dssalp02-HB,dssalp02
Error: Unable to resolve all hostnames.

I see you have a syntax error while creating the cluster with that command because the hostnames *should not* be separated by commas; rather you should separate them by blank spaces. Instead, please try to use the following if you are trying to create a cluster with 4 nodes:

pcs cluster setup --name dssa_cluster dssalp01-HB dssalp01 dssalp02-HB dssalp02

For additional details, please see the documentation about pacemaker on RHEL 6:

Red Hat Enterprise Linux 6 - Configuring the Red Hat High Availability Add-On with Pacemaker
https://access.redhat.com/documentation/en-US/Red_Hat_Enterprise_Linux/6/html-single/Configuring_the_Red_Hat_High_Availability_Add-On_with_Pacemaker/index.html

Please, let us know if the solution works and we can eventually close the case. Thank you very much.
