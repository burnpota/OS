SDS 상암센터 김지민 선임입니다.

현재 ECC AP 운영서버 1대의 physical disk의 예를 들면 아래와 같은 상황인데요.
처음에 volume 구성 시 fdisk를 통해서 LVM으로 partition을 하는 것과 그냥 disk를 사용하는 것의 차이점이 혹시 있는지에 대해서 문의드리려고 합니다.

아래 pvs 정보에서는 현재 /dev/mapper/SAP100G001 구성이 되어있는데 /dev/mapper/SAP100G001 p1(type:LVM)으로 구성할 때와의 차이점에 대해서 문의드리려고 합니다.


root@PECCAL01SL /root # pvs

/dev/mapper/DMP100G001 vg9 lvm2 a-- 100.00g 0

/dev/mapper/DMP100G007 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G008 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G009 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G010 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G011 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G012 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G013 vg9 lvm2 a-- 100.00g 0
/dev/mapper/DMP100G014 vg9 lvm2 a-- 100.00g 0
/dev/mapper/SAP100G001 LRPECCSAPVG lvm2 a-- 100.00g 77.99g
/dev/mapper/SAP100G002 LRPECCSAPVG lvm2 a-- 100.00g 0

What are the advantages and disadvantages to using partitioning when using LVM on LUNs?
