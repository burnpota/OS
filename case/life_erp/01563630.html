제목 : Ethernet Bonding over Bonding 구성 문의

======================
사전문의

어떤 문제/오류/결함이 발생했습니까? 기대하시는 결과는 무엇입니까?

안녕하세요.

먼저 아래와 같은 이더넷 본딩을 구성하려고 합니다.

+----eth0
+----bond0+----eth1
bond2
+----bond1+----eth2
+----eth3

bond0, bond1 은 mode4 (802.3ad aka LACP)를 이용하여 묶은 후 이 2개의 Bonding Device 를 다시 bond2 (mode active-backup) 으로 구성하는 방식입니다.

찾아보니 https://access.redhat.com/solutions/24528 에 이와 같은 방식이 지원이 안되는것 같은데요.

그렇다면, RHEL6 RHEL7 모두 지원이 안 되는 것인지요?

추가적으로 bonding mode 4 (802.3ad aka LACP) 를 사용할 때, https://access.redhat.com/solutions/24528 솔루션에 있는 방법처럼

4 개의 port 를 묶어 상위 스위치에서 2개씩 분리하는 방안이 적합한 방안인지 확인이 필요합니다.

.-----------. .-----------.
| Switch1 |---| Switch2 |
'-=-------=-' '-=-------=-'
| | | |
| | | |
.-=----.--=---.---=--.----=-.
| eth0 | eth1 | eth2 | eth3 |
|---------------------------|
| bond0 |
'---------------------------'
