"No more mirrors to try" 에러로 미루어 보아 java repository 접근에 문제가 발생했을 것으로 예상됩니다.
발생할 수 있는 원인은 아래와 같습니다.

1. yum cache의 충돌.
2. repository의 URL 접근 불가.
3. 다른 media repository와의 충돌.

문제 해결을 위해 아래 항목들을 통하여 진단해보시기 바랍니다.

1. yum clean all 이후 다시 설치 시도.
2. /etc/yum.repos.d/ 에 존재하는 *.repo 파일중에 java repository가 존재하는 파일을 찾아서 해당 baseurl에 접근이 가능한지 확인.
3. 별도로 만들어진 media repository가 존재한다면 해당 repository를 비활성화 후 설치 시도.

다음은 먼저 문의하신 java-1.8.0-openjdk-headless-* 패키지 설치 실패 이후의 영향도는 다시 설치하신다면 문제없이 작동될 것 입니다.
