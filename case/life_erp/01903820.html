Red Hat Global Support Services를 이용해주셔서 감사합니다.

12289 = 8192 + 4096 + 1

= 8192 O
TAINT_UNSIGNED_MODULE
Unsigned module has been loaded.
+ 4096 E
TAINT_OOT_MODULE
Out-of-tree module has been loaded.
+ 1 P/G
TAINT_PROPRIETARY_MODULE

P: A module with a non-GPL license has been loaded (this includes modules with no license). This significantly reduces Red Hat Global Support Services ability to provide effective support.
G: the kernel is tainted (for a reason indicated elsewhere), but all module code is licensed under the GNU General Public License (GPL) or a compatible license.

해당 taint 값은 위의 세개의 값의 합으로 산출되었습니다. 상기의 해당 값의 설명과 같이 커널이
unsigned module 과 out-of-tree module 그리고 non-GPL 라이센스의 모듈이 로딩되면서
taint된것을 의미합니다. 다시말하면 레드햇이 아닌 third-party의 모듈이 로딩되었음을 의미합니다.

[참고문서]
Why is the kernel "tainted" and how are the taint values deciphered?
-https://access.redhat.com/solutions/40594
