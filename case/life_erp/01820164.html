현재 사용 하신 kdump target은 vxfs 로 보여 supported kdump target 에
포함 되지 않습니다.

kdump.conf:
path /CRASH

fstab:
/dev/vx/dsk/DMPDG/CRASH /CRASH vxfs defaults,nofail,_netdev 1 2


현재 supported kdump target 에 대해 아래와 같은 링크에서 확인 가능 하시며
참고 부탁드립니다.

https://access.redhat.com/documentation/en-US/Red_Hat_Enterprise_Linux/7/html/Kernel_Crash_Dump_Guide/sect-supported-kdump-targets.html
