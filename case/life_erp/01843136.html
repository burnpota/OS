안녕하세요

Red Hat Global Support Services 를 이용해 주셔서 감사합니다.

여러 방법이 존재합니다.

1. rc.local에 아래 항목 추가
/sbin/ifconfig bond0 txqueuelen 10000 혹은 /sbin/ip link set bond0 txqueuelen 10000
- systemctl start rc-local
- chmod 755 /etc/rc.local

2. udev에 추가
vi /etc/udev/rules.d/60-custom-txqueuelen.rules
KERNEL=="bond[0-9]", RUN+="/sbin/ip link set %k txqueuelen 10000"

3. /sbin/ifup-local 생성
#!/bin/bash
if [ "$1" == "bond0" ]; then
/sbin/tc qdisc add dev "$1" root pfifo
/sbin/ip link set dev "$1" txqueuelen 10000

3번은 NetworkManager 를 비활성화해야합니다.
# systemctl disable NetworkManager
# systemctl stop NetworkManager

감사합니다.
