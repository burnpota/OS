안녕하세요,
Red Hat Technical Account Manager 한진구 입니다.

2016년 3월 25일 공개된 CVE-2016-0636 관련 안내입니다.


* CVE 버젼: CVE-2016-0636, https://access.redhat.com/security/cve/CVE-2016-0636
- 대상시스템: Red Hat에서 제공하는 openjdk JVM 버젼 7,8을 사용하는 제품 및 파생제품
- 관련패키지: java-<version>-openjdk


* 설명
- Hotspot 컴포넌트에서 improper type을 saftety 체크를 하는 부분의 이슈로, 신뢰되지 않은 Java application 또는 applet이 sandbox를 통과하지 않고 bypass하여 시스템 내부 자원을 사용할 수 있게 되는 보안 이슈입니다.
- 공격자는 특수하게 조작된 웹 사이트 방문을 통해 다운로드 파일을 유도하여 악성코드 유포 가능해 질 수 있습니다.


* 영향도

1. 심각도: 심각 (Critical)


2. 영향조건
- Red Hat에서 제공하는 openjdk Java 7/8 버젼을 사용하는 환경
