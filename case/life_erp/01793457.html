안녕하세요

삼성생명 ERP infra 지원 담당 SDS 금융시스템 그룹 김지민 선임입니다.

Overprovisioning이 적용된 스토리지에서 lun을 할당받아 사용 중인 OS에서 해당 영역 정리하는 방법을 문의드립니다.

스토리지는 Hitachi VSP G1K를 사용하고 있고
Overprovisioning이 적용된 영역에서 LUN을 할당 받아서 사용 중입니다.

Hitachi의 권고는 OS에서 zero page 처리를 한 후
스토리지에서 해당 LUN에 대해서 zeor page 처리를 수행하라는 권고를 받았는데

Redhat OS에서의 LUN에 대한 zero page 처리는
혹시 dd를 통한 /dev/zero 입력인지 문의드리려고 합니다.

감사합니다.

안녕하세요,

Red Hat Global Support Service 를 이용해주셔서 감사합니다.

Hitachi 에서 언급한 zero page 가 디스크 블록마다 모두 0 값을 넣어서 디스크를 초기화 하는 작업이 맡다면 아래의 dd
예제 명령어로 가능합니다.

$ dd if=/dev/zero of=/dev/sdx bs=xx count=xxxxx

혹시, Hitachi 에서 권고한 특정 명령어나 문서가 있으시면 좀 더 정확한 안내가 가능할거 같습니다.

감사합니다.
