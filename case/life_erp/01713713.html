Q1) 위에는 정책이 2개 나와 있고 각각 lun 하나씩 표기되어 있고 아래에는 정책이 1개 보이고 2개의 lun이 있는데요.
이 차이가 무엇인지요?
A1) 정책이 2개가 나와 있는것은 failover 모드이고, 1개가 보이는것은 multibus 모드입니다. - 참고문서[1]

Q2) 그리고 status값이 active와 enabled의 차이는 무엇인지 설명 부탁드릴게요. 아래의 결과 값은 테스트용인데요.
한쪽은 active이고 다른 한쪽은 enabled입니다.
A2) active 는 현재 I/O request 를 받고 있는 Path Group 이고, enabled 는 만약 active path group 내에
ready 상태인 path 들이 없을때 I/O request 들이 이동할 Path Group 입니다.

[ 참고문서 ]

[1] What is the difference between multibus mode and failover mode in device-mapper-multipath?
- https://access.redhat.com/solutions/3178
