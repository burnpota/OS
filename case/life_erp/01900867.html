안녕하세요,

Red Hat Global Support Services를 이용해주셔서 감사합니다.

해당 이슈는 인터널 버그 1403552에 리포팅되어 RHEL7.4 버전에서 픽스할것으로 확인 됩니다.

workaround로 디바이스 명칭 sda으로도 함께 blacklist를 진행하는것 입니다.

blacklist {
wwid 3600508b1001cb507acf85d4bbafb36c4
devnode "^(ram|raw|loop|fd|md|dm-|sr|scd|st)[0-9]*"
devnode "^hda[1-9]"
devnode "^sda[1-9]"

}
상기와 같이 적용해 주시고 reload 작업후 이슈가 해소되는지 확인 부탁 드립니다.

# systemctl daemon-reload
# systemctl reload multipathd.service

감사합니다.
