안녕하세요,

Red Hat Global Support Services를 이용해주셔서 감사합니다.

가능하시면 멀티패스 디바이스위에 있는 파일시스템은 umount 하신후 아래의 문서에
따라 alias를 변경해 주시고 기존에 있던 디바이스 삭제, 멀티패스 reload
작업을 해주시면 됩니다.

- 설정파일 변경
# cat /etc/multipath.conf
Example:
multipaths {
multipath {
wwid "3600123456789abcdef0123456789abc1"
alias "db_01"
}
}
- Flush 작업:
# multipath -f <multipath_device_name>

- Reload 작업
# systemctl reload multipathd
# multipath -v2
# multipath -ll <new name>

참고문서:
How to rename multipath (mpath) device names ?
https://access.redhat.com/solutions/66254
