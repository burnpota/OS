안녕하세요,
Red Hat Technical Account Manager 한진구 입니다.

2016년 12월 06일 공개된 CVE-2016-8655 관련 안내입니다.
현재 안내하는 보안 취약점은 Red Hat Enterprise Linux 5,6은 해당 되지 않습니다.

* CVE 버젼 : https://access.redhat.com/security/cve/CVE-2016-8655
- 대상 버전 : Red Hat Enterprise Linux 7 및 파생 버젼
- 관련 패키지 : kernel

이 보안 취약점은 TPACKET_V3 링 버퍼 생성 시 동기화를 처리하는 리눅스 커널 네트워킹 서브시스템 내의 Raw 패킷 소켓 구현에서 use-after-free 결점으로 이어지는 루트의 권한을 획득 가능한 보안이슈 입니다.

raw 패킷 소켓을 오픈할 수 있는 로컬 사용자(CAP_NET_RAW가 요구됨)가 시스템에서 root 권한 획득을 통해, 권한이 없는 로컬 사용자가 타겟 시스템에서 해당 익스플로잇을 사용해 DoS (서버 다운)을 야기시키거나, 관리자 권한으로 임의의 악성코드를 실행이 가능합니다.

영 향 도

1) 심각도 등급 : 중요 (Important)

2) 영향 조건

- kernel package 업데이트

3) RHEL 7
- 패치 진행 중 입니다.

해당 패치는 시스템의 Downtime을 요구하는 작업이며, 현재 해당 보안 취약점은 원격이 아닌 로컬에서 루트의 권한을 획들할 수 있는 보안취약점이기 때문에 서비스에 영향이 없도록 업데이트의 일정을 계획하시어 해당 패치버젼으로 업데이트 하는 것으로 권장 드립니다.

또한 향후 RHEL7에 대한 패치가 Release되는대로 바로 다시 안내드릴 수 있도록 하겠습니다.


감사합니다.
