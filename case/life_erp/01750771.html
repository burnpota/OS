안녕하세요,
Red Hat 한진구 입니다.

우선 SHMMAX, SHMALL 값은 커널에서 기본값이 적용되어 있으며, 설치 패키지중 initscripts 패키지를 설치하게 될 경우 RHEL6의 경우 그 기본값을 아래와 같이 변경하게 됩니다.

1. RHEL6 32bit default,


Default values are set in the default /etc/sysctl.conf

# Controls the maximum shared segment size, in bytes
kernel.shmmax = 4294967295

# Controls the maximum number of shared memory segments, in pages
kernel.shmall = 268435456

If these are removed the values default to

# cat /proc/sys/kernel/shmall /proc/sys/kernel/shmmax
2097152
33554432
======================
2. RHEL6 64bit default

Default values are set in the default /etc/sysctl.conf

# Controls the maximum shared segment size, in bytes
kernel.shmmax = 68719476736

# Controls the maximum number of shared memory segments, in pages
kernel.shmall = 4294967296


If these are removed the values default to

# cat /proc/sys/kernel/shmall /proc/sys/kernel/shmmax
2097152
33554432
======================
3. RHEL7 default

# cat /proc/sys/kernel/shmall /proc/sys/kernel/shmmax
18446744073692774399
18446744073692774399


참고로 SHMMAX는 byte단위이며, SHMALL은 페이지 단위 입니다.
======================

감사합니다.
