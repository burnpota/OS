Red Hat Global Support Service 를 이용해주셔서 감사합니다.

RHEL 7.2 버전을 기준으로 설명드립니다.

kernel-3.10.0-327.53.1.el7.x86_64

- kernel : Package Name
- 3.10.0 : Upstream version
- 327.53.1 : RPM Release Version
- el7 : RHEL Major Release
- x86_64 : Architecture

간단하게 말씀드리면 327.53.1 부분은 RPM Release version 을 의미하며 패키지가 변경됨에 따라 변할 수 있습니다.
RHEL 패키지의 의미에 대해서는 첨부 문서 [1] 에서 찾으실 수 있습니다.

[1] What is the meaning of the name for RPM package in Red Hat Enterprise Linux?
https://access.redhat.com/solutions/44508
