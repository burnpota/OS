안녕하세요,

Red Hat Global Support Services를 이용해주셔서 감사합니다.

서버에 설치되는 i686와 x86_64 아키택쳐의 패키지들의 버전은 일치해야 됩니다.
해당 이슈는 말씀하신바와 같이 RHEL 6.6 DVD 있는 버전보다 높은 패키지 버전들이 OS에 설치되어 있어
i686 패키지를 설치시 x86_64 버전과 일치하지 않아 발생된 이슈입니다.

해당 경우 아래와 같은 방법으로 이슈를 해소하실수 있을것으로 보입니다.

방법 1: 외부망으로 접속이 될 경우 서버를 저희 RHN에 등록하여 최신버전으로 업데이트 함

# subscription-manager register
# subscription-manager attach --auto
# yum install libcurl.i686

방법 2: RHEL6.7 DVD로 높은 버전으로 업데이트 함.

https://access.redhat.com/downloads/content/69/ver=/rhel---6/6.7/x86_64/product-software

방법 3: 현재 설치되어 있는 높은 버전의 패키지들을 RHEL 6.6 DVD에서 제공한 패키지로 downgrade 함.
주의사항: 해당 방법을 사용시 앞서 어떠한 버그 픽스에 의하여 업그레이드된 패키지도 함께 downgrade 될수
있으니 추천하는 방법은 아닙니다.

감사합니다.
