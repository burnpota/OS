/dev/mapper/vg1-lv_ORACLE on /oracle type ext3 (rw)
/dev/mapper/vg1-lv_HIWARE on /hiware type ext3 (rw)
/dev/mapper/vg2-oradata_STG on /oradata_STG1 type ext4 (rw)
/dev/mapper/vg2-oraarch_STG1 on /oraarch_STG1 type ext4 (rw)
none on /proc/sys/fs/binfmt_misc type binfmt_misc (rw)

/oracle 과 /hiware을 ext3포멧에서 ext4포멧으로 변경을 하고 합니다.
아래의 작업이 맞는지 확인 부탁드리며,
작업간 데이타에 영향도가 있는지 코멘트 바랍니다.

[작업]
umount /oracle
umount /hiware

tune2fs -O extents,uninit_bg,dir_index /dev/mapper/vg1-lv_ORACLE
tune2fs -O extents,uninit_bg,dir_index /dev/mapper/vg1-lv_HIWARE

e2fsck -fDC0 /dev/mapper/vg1-lv_ORACLE
e2fsck -fDC0 /dev/mapper/vg1-lv_HIWARE

vi /etc/fstab
ext3부분을 ext4로 변환

mount -a

확인부탁드립니다.
