안녕하세요

Red Hat Global Support Services 를 이용해 주셔서 감사합니다.

아래 문서에 요청하신 iso 를 통해서 레포지터리를 구성하는 방법이 기술 되어있습니다.

Need to set up yum repository for locally-mounted DVD on Red Hat Enterprise Linux 7
https://access.redhat.com/solutions/1355683

문서를 기반으로 설명드리자면,


iso 파일을 특정 위치에 다운로드하고, 마운트를 한 상태로 유지합니다.

# mount -o loop /root/RHEL7.4.iso /mnt

마운트된 디렉토리에서 레포지터리 파일을 복사합니다.

# cp /mnt/media.repo /etc/yum.repos.d/rhel7dvd.repo
# chmod 644 /etc/yum.repos.d/rhel7dvd.repo

복사한 파일을 편집하여 아래와 같이 설정합니다.

enabled=1
baseurl=file:///mnt/
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-redhat-release

아래 명령을 수행하여 레포지터리 캐시를 삭제합니다.
# yum clean all

아래 명령을 수행하여 시스템에 설치된 패키지들을 iso 버전으로 모두 업데이트합니다.
# yum update

만약 본 이슈와 관련하여 추가적인 문의 사항이 있으실 경우 연락 주시기 바랍니다.

감사합니다.
