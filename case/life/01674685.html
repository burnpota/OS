useradd할시에 생성된 유저의 owner 를 결정하는 config file는 어디서 설정이 가능한가요?

예를들면
useradd test 하면
ls -al /home/test
rwx------ 2 test test 4096 2016-07-26 05:18 .
drwxr-xr-x. 4 root root 4096 2016-07-27 00:00 ..
-rw------- 1 test test 765 2016-07-13 01:37 .bash_history
-rw-r--r-- 1 test test 18 2014-09-26 19:29 .bash_logout
-rw-r--r-- 1 test test 176 2014-09-26 19:29 .bash_profile
-rw-r--r-- 1 test test 124 2014-09-26 19:29 .bashrc
owner이 test group test로 되어있는데 이것을 결정하는 설정파일은 어디 있나요?
퍼미션은 /etc/login.def
베이스디렉토리는 /etc/default/useradd 인것은 알고 있는데
owner의 설정파일은 알수가 없습니다.
