안녕하세요,

Red Hat Global Support Service 를 이용해주셔서 감사합니다.

Build date 는 알고 계신것 처럼 패키지가 빌드된 날짜입니다.
패키지는 빌드가 되면 QA 를 거친 이후에 릴리즈 됩니다.
따라서 Build Date 와 Release Date 는 관련이 없으며, 실제로 다른 것 입니다.

다음은 패키지의 Release Date 를 확인하는 방법입니다.
아래 Errata Page 를 방문하셔서 해당 패키지를 검색하면 해당하는 Advisory 가 나열되며 Publish Date 를 확인하시면 됩니다.

"Red Hat Product Errata"
https://access.redhat.com/errata/#/

문의주신 두 패키지의 Release date 와 Advisory 는 아래와 같습니다.

- kmod-20-15.el7_4.7 / RHBA-2018:0159 / 2018-01-25
- kmod-20-21.el7 / RHBA-2018:0799 / 2018-04-10
