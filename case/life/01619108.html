안녕하세요,

Red Hat Global Support Services 를 이용해 주셔서 감사합니다.

1. 이슈가 발생된 서버 버전이 RHEL6.8 beta 이신가요?
맞다면 beta 버전은 저희 지원 범위가 아님을 알려 드립니다.

2. 기타 서버 버전일 경우 말씀하신 코어 분석은 kdump 에 의하여 생성된 vmcore 파일인지
아니면 특정 프로그램의 코이 인지 확인 부탁드립니다.

3. vmcore 라면 /etc/kdump.conf 파일을 확인하여 vmcore가 발생되
디렉토리에서 관련 파일을 저희 dropbox[1]에 올려주시고 파일명을 알려주시기
바랍니다.

4. 프로그램의 core라면 어느 프로세스 밎 해당 프로세스를 제공하는 패키지 버전은
어떻게 되는지 확인 부탁 드립니다.

참고문서:
