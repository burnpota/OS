Context 는 CPU 의 register 와 program counter 내에 있는 내용이고, Context Switch 는 커널이 이들을 사용하는
메모리내의 프로세스에 필요로한 CPU 의 진행상태를 보류하거나, 프로세스를 재개하기 위해서 program counter 가 지시하는 위
치로 반환시키거나 CPU 레지스터내에 다시 이 상태들을 저장하고, 메모리로부터 다음 프로세스의 context 를 회수하는 활동을 의
미합니다.

쉽게 말하자면, 커널이 CPU 상의 한 프로세스의 실행을 보류하고, 이전에 보류되어진 어떤 다른 프로세스의 실행을 재개해는 활동
이라고 할 수 있습니다.

그래서, cswch/s 는 초당 위의 활동을 한 횟수로 이해하실 수 있습니다.

감사합니다.
