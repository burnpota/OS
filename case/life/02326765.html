안녕하세요. Red Hat 김마현입니다.

관련하여 내용 업데이트해 드리도록 하겠습니다.

issue : 로컬(콘솔)에서 로그인 시 표시하는 메시지. 기본은 OS 버전정보를 출력. 일반적으로 운영환경에서는 아무 내용도 넣지 않음.
issue.net: 리모트(telnet)에서 로그인 시 표시하는 메시지. 기본은 OS 버전정보를 출력. 일반적으로 운영환경에서는 아무 내용도 넣지 않음.
motd : 로그인 시 표시되는 메시지를 기술하는 파일(로컬, 리모트 모두 포함). OS 기본은 blank 임.
일반적으로 운영환경에서는 이 파일에 다음과 같은 경고 문구를 넣음 "이 시스템은 허가되지 않은 접속자에게........"

위 파일은 단순히 서버 접속자에게 전달하고 싶은 내용을 디스플레이하는 파일로 동시 사용이 가능합니다.
사용 방법은 해당 파일을 vi 편집기로 해당 파일을 편집하면 됩니다.

감사합니다.
