안녕하세요,

RHEL에서 support 하는 Apache HTTP Server Version은 아래와 같습니다.

Apache HTTP Server versions included in currently-supported products
Red Hat Enterprise Linux (RHEL)1 httpd rpms
RHEL 5: based on upstream v2.2.3
RHEL 6: based on upstream v2.2.15
RHEL 7: based on upstream v2.4.6

이에 대한 보다 자세한 내용은 [1]을 참조하실 수 있습니다.

RHEL의 Apache는 community 버전의 Apache를 기반으로 하지만 정확히 동일한 것은 아닙니다.
Additional notes 부분에서 아래와 같은 설명을 보실 수 있습니다.

~~~
For example: EWS 2.1.0 & EAP 6.4.0 include Apache httpd based on upstream v2.2.26; however, they also include multiple CVE security fixes which are not in the original community release of Apache httpd 2.2.26
~~~

즉, RHEL의 Apache HTTP Server의 경우 community 버전의 upstream을 기반으로 하고 있지만 CVE security fix를 더 포함하고 있습니다.
또한 추가적으로 발생하는 보안이슈에 대해서 계속적으로 support가 되고 있습니다.

따라서 보안 이슈가 있다면, 어떤 것이 있는지(CVE 번호 명시) 새로이 case를 오픈해서 문의해 주시면 되겠습니다.

참고로, Apache httpd의 community release는 support 되지 않습니다.


[1] https://access.redhat.com/solutions/445713
