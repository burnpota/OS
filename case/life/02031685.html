안녕하세요? 레드햇 허경입니다.

USB 관련 메시지는 iDRAC을 통하여 Virtual Console을 실행할 때 발생하는 것을 저희 테스트 시스템에서 확인하였습니다.
iDRAC의 Virtual Console을 실행하면 키보드와 마우스를 USB로 인식하게 됩니다.

Virtual Console 프로그램의 Virtual Media 메뉴에서 Connect Virtual Media 메뉴를 클릭하면 아래와 같이 USB 디스크 장치가 인식되는 것으로 보이게 됩니다.

[root@dell-per730-12 ~]# lsusb
Bus 002 Device 002: ID 8087:8002 Intel Corp.
Bus 002 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
Bus 001 Device 005: ID 0624:0251 Avocent Corp. Virtual Mass Storage <=== Virtual Media Connected
Bus 001 Device 004: ID 0624:0249 Avocent Corp. Virtual Keyboard/Mouse
Bus 001 Device 003: ID 413c:a001 Dell Computer Corp. Hub
Bus 001 Device 002: ID 8087:800a Intel Corp.
Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub

다시 Virtual Media 메뉴에서 Disconnect Virtula Media를 클릭하면 아래와 같이 Virtual Mass Storage 가 연결해제 됩니다.

[root@dell-per730-12 ~]# lsusb
Bus 002 Device 002: ID 8087:8002 Intel Corp.
Bus 002 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
Bus 001 Device 004: ID 0624:0249 Avocent Corp. Virtual Keyboard/Mouse
Bus 001 Device 003: ID 413c:a001 Dell Computer Corp. Hub
Bus 001 Device 002: ID 8087:800a Intel Corp.
Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub

/var/log/messages 로그에도 보내주신 sosreport에 기록된 것과 같은 로그가 기록됩니다.

아래와 같은 방법으로 확인하실 수 있습니다.
1. tail -f /var/log/messages 명령으로 로그 파일을 열어 지켜봅니다.
2. iDRAC의 Virtual Console로 접속하셔서 Virtual Media를 연결합니다.

확인해보시고 결과를 알려주시기 바랍니다.
