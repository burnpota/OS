다만, 다른 시스템도 그렇고 RHEL7.1에서 RHEL7.2로 문제없이 업데이트된 것을 기준으로 본다면 현재 시스템의 구성 정보가 이전과 달리 별도의 설치된 버젼이 있는건 아닌가 싶은 의심이 한가지 있고,

그래서 우선 아래와 같이 작업 을 시도해 주시기 바랍니다.


1. # yum clean all
2. # yum repolist -> 해당 repo정보를 통해서 repository가 정상적으로 RHEL7.2 repository가 등록된지를 확인바랍니다.
3. # yum update --disablerepo=* --enablerepo=<repo명>

이후 문제가 지속적일 경우, sosreport를 수집하여 업데이트 바라며 추가적으로 아래 정보도 확인 바랍니다.

# yum repolist > yum_repolist.txt
