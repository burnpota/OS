Red Hat Global Support Services를 이용해주셔서 감사합니다.

아래와 같은 flag는 해당 CPU가 HT(Hyper-treading)를 지원함을 알리는 정보이며 이와 같은 메시지로
해당 서버가 HT 기능 활성화 여부는 알수 없습니다.
HTT (Multi-threading)

해당 서버에서 HT가 기동 여부는 /proc/cpuinfo 출력 결과중 siblings 와 cpu cores 값이
동일한지에 의하여 판단하실수 있으십니다. 이 두 값이 동일하면 HT가 비활성화되어 있는것이며
siblings 값이 cpu cores 의 배수의 경우 HT가 활성화되어 있습니다.

따라서 현재 서버에서는 HT가 활성화되어 있는것으로 판단됩니다.

감사합니다.

현재 서버에 하이퍼 스레딩이 ON상태입니다.
시스템 상에서 하이퍼 스레딩 관련 확인 할 수있는 부분이
1. dmidecode 상에서 HTT 가 Multi-threading면 OFF
HTT 가 Hyper-threading technology 면 ON

2. cat /proc/cpuinfo 에서 siblings 와 cpu cores 가 같으면 OFF
sibling가 cpu cores의 두배면 ON으로 알고 있습니다.

그런데 아래 로그를 보시면
HTT 가 Multi-threading 임에도 불구하고
sibling가 cpu cores의 두배입니다.
