예를 들어 ID 11 에 대해 undo와 rollback을 각각 수행 한다고 치면,

만약 ID 11에 대한 undo를 아래와 같이 실행하면 2개의 패키지가 uninstall (즉, 제거)될 것입니다.
# yum history undo 11
--> 지정된 ID의 포함된 패키지들만 삭제

만약 ID 12에 대한 rollback을 아래와 같이 실행하면 3개의 패키지가 uninstall (즉, ID 12, 13의 패키지들 삭제)될 것입니다.
# yum history rollback 11
--> 지정된 ID이후의 패키지들만 삭제(ID 11은 제외)


아래는 관련된 Man page의 내용입니다.
For example, if you have 3 transactions, where package A; B and C where installed respectively.
Then "undo 1" will try to remove package A, "redo 1" will try to install package A (if it is not still installed), and "rollback 1" will try to remove packages B and C.
