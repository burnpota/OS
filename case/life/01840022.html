1. 현재 오라클 과 리스너 start 와 stop을 ocf:heartbeat의 agents를 사용하고자 합니다.
pacemaker에 등록하는 방법에 대해 알려주시기 바랍니다.

RHEL7 pacemaker 환경에서 사용 가능한 oracle DB 버전은 11g, 12c 이며
등록 방법은 예를 들어 아래와 같습니다.

# pcs resource create db oracle sid=ttdb home=/ora/db user=oracle --group oracle_group
# pcs resource create listener oralsnr sid=ttdb home=/ora/db user=oracle --group oracle_group

참고문서:
Can Oracle Database 10g, 11g, and 12c be managed by a Red Hat High Availability cluster?
https://access.redhat.com/solutions/25970

2. pacemaker의 리소스 그룹이 서버를 reboot후에는 사용자가 수동으로 start 하는 방법을 사용하고자 합니다. 그러나 failover시에는 리소스 그룹이 자동으로 start되게 하고자 합니다. 이와같은 방법이 가능한지 가능하다면 어떠한 방식으로 구성을 해야하는지 확인 부탁드립니다.

상기와 같은 기능은 pacemaker에 없는것으로 확인됩니다.
