현재 세션 실패 현상중 특이한 부분이 있어 관련 조치 설정이 있는지 확인을 부탁드립니다.



https 로 세션은 유지되다가 http 로 같은 url 호출시 세션이 리셋되는 현상이 있습니다.

https://neofussvr.samsung.com.cn/monitor/test.jsp 최초 호출시 세션 유지됨

http://neofussvr.samsung.com.cn/monitor/test.jsp 이후 http 로 같은 URL 호출시 세션이 유지되지 않고 실패함.



반대로 http 로 최초 요청후 https 로 요청시에는 세션이 유지되는 것으로 확인됩니다.



https -> http 로 프로토콜 변경시 세션이 유지 안되는 현상에 대한 해결 설정이 있는지 확인 부탁드립니다.
