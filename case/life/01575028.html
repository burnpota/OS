안녕하세요
아래의 RHEL7 관련 명령어 대해서 확인 부탁드립니다.
RHEL7 관련 아래 명령어를 수행 후 설정 변경 부분이 있는지 확인 부탁드립니다.

1. RHEL7 의 grub.conf 파일 복사
cp /boog/grub2/grub.conf /tmp/sos_tg/boot/grub.conf

2. RHEL7 boot 시 시작하는 Service들 확인 후 로그에 저장
systemctl list-unit-files >> /tmp/sos_tg/sos_commands/startup/systemctl_list-unit-files

3. RHEL7 boot 시 시작하는 Service들 추가 확인 후 로그에 저장
systemctl list-units >> /tmp/sos_tg/sos_commands/startup/systemctl_list-units

4. RHEL7 Filesystem Type 중 xfs 에 대한 정보 확인후 로그에 저장
xfs_info / >> /tmp/sos_tg/sos_commands/filesystem/xfs_info
xfs_info /boot >> /tmp/sos_tg/sos_commands/filesystem/xfs_info
xfs_info /home >> /tmp/sos_tg/sos_commands/filesystem/xfs_info
